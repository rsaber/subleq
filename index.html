challenge #1
puzzle by riyasat

Your first challenge is to write a program that adds two numbers together. The input is in a,b and the result must be stored in c

editor

instruction set

subleq

subtracts contents at address a from contents at address b, stores result at address b, if result is not positive move program counter to address c

That's right. Only one instruction. And it's Turing complete. Good luck.
